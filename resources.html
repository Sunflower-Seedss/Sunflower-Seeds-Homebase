Aster (V2.1)

Chromium Extension

I may have gone overboard! 😅 V1.2 was fun, but now we have a bunch of Creator tools, too. It's fully customisable—hide the tools you won't use, choose between color themes and light/dark modes, and scale it however you like.

✨ Fun fact: Aster is named for the genus in the Asteraceae family—the same family that includes sunflowers and daisies, two of my favourite flowers!

💬 Chat Tools — Stats, Nexus reminder, save regenerations, user input recovery, auto-refresh on stop, delete thinking blocks, scroll to top/bottom, and download your chats as .txt files. Plus Quill's message improvement and chat summarization.

🛠️ Creator Tools — Export/import bots as .json with all toggles and tags. Lorebook testing, active chat scanner, token budget counter, and a workshop for merging and managing lorebooks. Plus Quill's character lens and import from other platforms.

✒️ Quill — Connect your own LLM (Ollama, LM Studio, koboldcpp, or any OpenAI-compatible API). The output varies based on what you have attached, so read the guide before using it.

🔒 Privacy: Everything runs locally. All data stays in your browser's local storage unless you explicitly send text to a model you've connected.

Sunny’s DreamJourney Toolkit

Chromium Extension · V1.2 (Archived)

Stats — tracks message counts and rerolls per chat; your messages, the bot’s, and individual rerolls as separate stats.

Nexus Reminder — shows how many messages have passed since you last checked your Nexus. Turns orange then red as you get further from it. Ignore completely if the Nexus isn’t your thing.

Save Regenerations — saves previous bot replies before they disappear on regen. Browse them in the panel and swap one back in with one click.

User Input Recovery — autosaves what you’re typing so a crash won’t lose your message. Also keeps your last 5 sent messages, if Stop Generation deletes yours, you can grab it back.

Auto-refresh on Stop — stopping mid-generation can leave duplicate or missing messages. Gives you a 3-second countdown to refresh and clear the error. Cancel it if you need to copy something first.

Delete Thinking (Nyx & Athena only) — permanently removes the thinking block from a message once you’re done with it.

Download Chat — exports the full conversation as a .txt file with the character’s actual name on each message. Hit it and cancel to jump to the top of a long chat without downloading.

Light/dark mode toggle built in. Help page at the bottom if you forget what anything does.

Browser support: Chrome, Edge, Brave, and any Chromium-based browser. Data is saved locally, stats persist between sessions unless you uninstall. No cross-session persistence on Firefox. Mobile: works with Lemur browser (separate download below).

How to install

  1. Download the zip and unzip it somewhere permanent. Don’t delete the folder after installing or the extension will break.
  2. Go to chrome://extensions in your address bar (Edge: edge://extensions).
  3. Turn on Developer mode using the toggle in the top right.
  4. Click Load unpacked and select the dreamjourney-toolkit folder inside the unzipped files.
  5. The sun icon should appear in your toolbar. If it’s hidden, click the puzzle piece icon and pin it.

Activates automatically whenever you open a DreamJourney chat. Other Chromium browsers work the same way.
